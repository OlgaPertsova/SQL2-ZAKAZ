8. Вывести коды всех продавцов заказчики, 
которых работают в городе Москва и не живут там.
SELECT NAME, KOD
FROM ZAKAZ
WHERE CITY = 'Москва' AND CITY2 != 'Москва';

9. Выведите в обратном порядке фамилии всех заказчиков, 
которых обслуживает продавец с номером 1012.
SELECT NAME
FROM ZAKAZ
WHERE KOD == 1012
ORDER BY NAME DESC;

10. Определите всех заказчиков, которые живут 
во Владимире, и не покупали кухонные стенки.
SELECT NAME, Prod, CITY
FROM ZAKAZ
WHERE CITY =='Владимир' AND Prod != 'кухонные стенки';

11. Определите в магазинах, каких городов продаются диваны 
этой фирмы. (Вывести атрибуты характеризующие город, товар и место продажи).
SELECT CITY, Prod, CITY2
FROM ZAKAZ
WHERE Prod =='диваны';


12. Вывести на первом месте код продавца, на втором фамилию 
заказчика. Код должен быть упорядочен в прямом порядке, а фамилии в обратном. 
SELECT KOD, NAME
FROM ZAKAZ
ORDER BY 1 ASC, 2 DESC;

13. Напишите запрос, который выведет коды всех продавцов 
заказчики, которые живут в Химках и покупают стулья.
SELECT KOD, NAME, CITY, Prod
FROM ZAKAZ
WHERE CITY =='Химки' AND Prod == 'стулья';

14. Определите города, в которых продают стулья или столы для ПК.
SELECT CITY, Prod
FROM ZAKAZ
WHERE Prod == 'стулья' OR Prod == 'столы для ПК';

15. Определите коды продавцов, работающих по доверенности с 
заказчиками, живущими в Москве и имеющими рейтинг менее  190 пунктов.
SELECT KOD, REM, CITY, RATING
FROM ZAKAZ
WHERE REM == 'доверенность' AND CITY == 'Москва' AND RATING < 190;
